Introduction

Introduction

Time Capsule is the applicaiton for a thing-to-do.

The application provides the differentiated interface for the registration of a thing-to-do.
It allows you to register, modify and delete things-to-do, not merely displaying the list.

It provides an easy interaction for you to register a thing-to-do by clicking and dragging on the Timeline;
a registered thing-to-do is displayed in the form of a bar graph.
