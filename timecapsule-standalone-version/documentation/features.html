Features

Features

TimeLine

The application provides an entirely different interface for things to do.
It's the Timeline.
The Timeline is an interactive space where you can select a time and register a thing to do.

Bar

Bar is created on the Timeline when you register a thing-to-do.
The size of the bar is exactly equal to the registered amount of time and the bar is the tool needed to manage your things-to-do.
You can see the simple content of a thing-to-do with a tooltip and are also able to mark the completion of a thing-to-do.
Bar also provides the interface for modification and deletion.

Timesheet

The digital clock just above Timeline displays current date and time.
And a registered or completed thing-to-do is indicated by numbers.

Timelist

The registerd things-to-do on the Timeline are listed by date and time.
The color of completed things-to-do is changed to gray.

Send Json

The application doesn't provide the ability to save things-to-do in a file.
However, you can export registered things-to-do in JSON formats and transfer them to the database.

Load Json

If you have data saved as JSON file, you can load it.
'Load Json' shows method which JSON file stored in database is loaded on the Timeline.
