Code settings

Code settings

Including resource files

To get started you need some resource files. You have to include the necessary resource files to your pages.

css

To use css files, you need to insert these code in the <head> tag of your HTML.

<!-- External stylesheet -->
<link rel="stylesheet" href="../css/bootstrap.css">
<!-- TimeCapsule stylesheet -->
<link rel="stylesheet" href="../css/common.css">

Js

Considering performance, you need to put the script tags at the footer of your pages before the closing </body> tag.

<!-- External libraries: jQuery & BootStrap -->
<script src="../js/lib/jquery.js"></script>
<script src="../js/lib/bootstrap.js"></script>
<!-- TimeCapsule script files -->
<script src="../js/app.combined.min.js"></script>

Including HTML

Timesheet

Timesheet is a container including the Timeline which is an interactive space to register a thing to do. Also, It has a clock displaying current date and time and shows count of registration and completion in numbers.

You neet to put the example code below within the <body> tag of a HTML.

<!-- Timesheet //-->
<div class="panel panel-bx">
	<div class="panel-time">
		<div id="clock">
			<span class="date">0000-00-00</span>
			<span class="time">00:00</span>
		</div>
	</div>
	<div class="panel-body">
		<div class="timesheet-bx">
			...
		</div>
	</div>
	<div class="panel-info">
		<ul>
			<li><a href="#" class="total">To-do <span class="badge">0</span></a></li>
			<li><a href="#" class="done">Complete <span class="badge">0</span></a></li>
		</ul>
	</div>
</div>
<!--// Timesheet -->

Timeline

You have to define an element for the Timeline within the Timesheet section of a page. The Timeline is an interactive space where you can select a time and register a thing to do.

Put the following code below within the <div class="timesheet-bx"> tag of a HTML.

<a href="#" class="prev" id="left"><span class="glyphicon glyphicon-chevron-left" aria-hidden="true"></span></a>
<div class="time-wrap">
	<div class="time-inner">
		<!-- Timeline //-->
		<div id="time-sheet">
			<div id="time-line"></div>
			<ol id="time-list" class="clfix"></ol>
		</div>
		<!--// Timeline -->
	</div>
</div>
<a href="#none" class="next" id="right"><span class="glyphicon glyphicon-chevron-right" aria-hidden="true"></span></a>

The Timeline code is directly created within the <div id="time-line"> tag when HTML page is loaded.

Timelist

The registerd things-to-do are listed by date and time. Put the following code below within a HTML.

<section class="content">
	<form>
	<fieldset class="form">
		...
		<!-- Timelist //-->
		<div class="todo-area" data-template="./html/time-list.tpl">
			<p class="nolist">No registered.</p>
		</div>
		<!--// Timelist -->
		...
	</fieldset>
	</form>
</section>

At first, It will be empty becase there is no registered. If you register a thing-to-do, it is listed within the <div class="todo-area"> tag.

data-template

Timelist use a template when it is loaded on page. Notice the data-template attribute. We will use its value as a template for the Timelist.

<div class="todo-area" data-template="./html/time-list.tpl">

Registration form

When you register a thing to do on the Timeline, An Modal popup is opened and you need to fill out a form.

<!-- Registration form //-->
<div class="modal fade" id="todoModal" tabindex="-1" role="dialog" aria-labelledby="todoModalLabel" aria-hidden="true">
		  ...
		  <form>
			  <div class="form-group">
				<label for="todo-title" class="control-label">Title :</label>
				<input type="text" class="form-control" id="todo-title">    
			  </div>
			  <div class="form-group">
				<p class="b">Date :</p>
				<p class="txt-time">
					<span class="glyphicon glyphicon-calendar" aria-hidden="true"></span>
					<span id="startDate">0000-00-00</span>
					<span class="glyphicon glyphicon-time ml5" aria-hidden="true"></span>
					<span id="startTime">00:00</span>
					<span class="txt-bar">~</span>
					<span class="glyphicon glyphicon-calendar" aria-hidden="true"></span>
					<span id="endDate">0000-00-00</span>
					<span class="glyphicon glyphicon-time ml5" aria-hidden="true"></span>
					<span id="endTime">00:00</span>
				</p>
			  </div>
			  <div class="form-group">
				<label for="todo-text" class="control-label">Description :</label>
				<textarea class="form-control" id="todo-desc"></textarea>
			  </div>
			</form>
			...
</div>
<!--// Registration form -->
