Easy Installation

Easy Installation

Time Capsule is a standalone application developed using Javascript, jQuery and Ajax.
Copy folder and paste to your server/localhost. That’s it installation done.

Important Note

Time Capsule uses a data of Json formats.
JSON(JavaScript Object Notation) is a lightweight data-interchange format.

It exchanges data in use of Ajax.

It means that to work well, you need your server or local server.

Test

  1. Download an item then extract zip files.
  2. For test, upload all files including json file to your own hosting server or localhost.
  3. Open index.html into a browser.
  4. Click 'Load Json' button in a page, then select a date.
  5. If bars are loaded on the Timeline and To-Do lists are listed, it's OK.
