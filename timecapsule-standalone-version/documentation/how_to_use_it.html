How to use it

How to use it

Register a thing to do

  1. Click the time you want to register on the Timeline.
  2. If a bar is created on the Timeline, move a mouse and click the end time.
  3. The registration modal popup is opened.

Registration popup

  1. The time you clicked on the timeline is displayed on the Date.
  2. Please enter your title and description. Title is required. If you want to register, please click the 'Register' button. Click the 'Cancel' button to cancel.
  3. When you click the 'Register' button, registration is completed with the message, 'It was registered'.
  4. When it is complete, the To-Do list is listed by date and time.

Edit and delete

There are buttons to edit and delete on the bar which is created on the Timeline.

Edit

  1. To edit the item, click the 'Edit' button on the bar.
  2. An edit popup similar to the registration popup opens.
  3. Edit the "Title", "Description" and click the "Edit" button.
  4. When completed in edit, to-do lists are updated.

Delete

  1. To delete the item, click the 'Delete' button on the bar.
  2. A message alert opens to confirm the deletion.
  3. If you want to delete, click the 'OK' button.
  4. The bar on the timeline is removed and the item is removed from the To-Do list.

View a tooltip

  1. When you mouse over a bar created on the timeline, a tooltip appears.
  2. A tooltip let you view the title, description, and time of an item.
  3. When you mouse out a bar, a tooltip disappears.

Change to Done

  1. When you register an item of to do, it will be set in progress and a green light will be lit on the bar.
  2. To change an item to Done, click the green lighted button on the bar.
  3. A message alert opens to confirm the completion.
  4. If you want it, click the 'OK' button.
  5. It changes to Done and the green button on the bar is set to gray.
  6. The completed item in the To-Do list turns gray.
